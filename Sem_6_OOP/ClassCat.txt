Задание No2
1. Продумайте структуру класса Кот. Какие поля и методы будут актуальны для приложения, которое является
а) информационной системой ветеринарной клиники
б) архивом выставки котов
в) информационной системой Театра кошек Ю. Д. Куклачёва
Можно записать в текстовом виде, не обязательно реализовывать в java.

а) Ветеринарная клиника.
Поля:
1. Имя кота (кличка) - name;
2. Возраст - age;
3. Вес - weight;
4. Порода - breed;
5. Владелец - Owner;
6. Прививки - vaccina;
7. Номер карты - IdHistoryMedical;
8. История болезни  - HistoryMedical;

Методы:
1. Геттеры - получаем значения для полей (имя кота, возраст, порода и т.д.);
2. Сеттеры - устанавливаем значения для всех полей;
3. add - добавляем записи в поля (addVaccina, addHistoryMedical);

б) Архив выставки котов.
Поля:
1. Имя кота (кличка) - name;
2. Возраст - age;
3. Вес - weight;
4. Порода - breed;
5. Владелец - Owner;
6. Прививки - vaccina;
7. Родословная - Pedigree;
8. Награды - Titul;

Методы:
1. Геттеры - получаем значения для полей (имя кота, возраст, порода и т.д.);
2. Сеттеры - устанавливаем значения для всех полей;
3. add - добавляем записи в поля (addTitul);

в) информационной системой Театра кошек Ю. Д. Куклачёва
Поля:
1. Имя кота (кличка) - name;
2. Возраст - age;
3. Вес - weight;
4. Порода - breed;
5. Окрас - Color;
6. Прививки - vaccina;
7. Дрессировщик - Trainer;
8. Навыки - Skills

Методы:
1. Геттеры - получаем значения для полей (имя кота, возраст, порода и т.д.);
2. Сеттеры - устанавливаем значения для всех полей;
3. add - добавляем записи в поля (addSkills);
